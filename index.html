[+]== WHO WE ARE ============================================================[+]
|                                                                              |
| We are a student-led club which focusses on the hands-on practical skills    |
| as well as theoretical foundations to Artificial Intelligence and            |
| developing the skills needed to get started. We welcome anyone with          |
| motivation to learn and passion to guide. We aim to deliver a wide range of  |
| modalities including Vision, Speech, Language and more.                      |
|                                                                              |
[+]==========================================================================[+]


[+]== WHAT WE DO ============================================================[+]
|                                                                              |
| At SoILS we offer curated learning environments, talks, competitions and     |
| more! We host weekly meetups to discuss ongoing research, potential          |
| collaborations and other chit-chat.                                          |
|                                                                              |
[+]==========================================================================[+]


[+]== MEMBERSHIP ============================================================[+]
